# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in `lphenom/redis`, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

### How to report

Send an email to: **popkovd.o@yandex.ru**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response time

We will acknowledge receipt within **48 hours** and aim to provide a fix within **7 days** for critical issues.

### Disclosure

We follow responsible disclosure — we will coordinate with you before publishing details of the vulnerability.
